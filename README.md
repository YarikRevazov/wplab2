## Ответы на контрольные вопросы

### 1. Что делает тема и что делает плагин?

Тема отвечает за внешний вид сайта: дизайн, цвета, расположение элементов.
Плагин добавляет функциональность, например формы, редакторы или дополнительные возможности.

---

### 2. Почему контент не теряется при смене темы?

Контент хранится в базе данных WordPress.
Тема влияет только на отображение, а не на сами данные.

---

### 3. Как изменить внешний вид без редактирования кода?

Можно использовать разделы:

* Внешний вид → Темы
* Внешний вид → Настроить

Там можно менять цвета, логотип, структуру и другие параметры без программирования.
